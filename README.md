## 🧠 Sobre mim

Sou **João Pedro**, estudante de **Sistemas de Informação (6º período - UNIT-SE)** e atuo como **Desenvolvedor Front-End com foco em UI/UX e experiências digitais modernas**.

Minha especialidade é construir interfaces que não apenas funcionam — mas **impressionam, convertem e resolvem problemas reais**.

Penso além do código:
